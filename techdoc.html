Java Entry Point

This entry point to a java application is the main function.

It is categotized by its (strings[] arg) as a parameter to thr function 

            
                public class MyMainFunction {
                    public static void main(String[] args) {
                        System.out.println("Hello, world!"); // Example: Prints "Hello, world!"
                        
                    }
                }

As you can see the main function is wrapped with a class which is part of the object oriented structure of Java Projects.

The name of the project is therefore "MyMainfunction"

Object Oriented Programming

Java is known as an object oriented programming language.

This means that it is easy to represent entities as objects by using classes and encapsulation.

An exammple of this might be a Student class to represent a student 

            
                public class Student {
                    /* Student properties */
                    private String name;
                    private int age;

                    /* Constructctor */
                    public Student(String name,int age){
                        this.name = name;
                        this.age = age;
                    }

                    /* Getter method */
                    public String getName(){
                        return name;
                    }

                    /* Setter method */
                    public void setName(String name){
                        this.name = name;
                }
              }
